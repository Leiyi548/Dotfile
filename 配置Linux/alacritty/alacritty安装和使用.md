# alacritty 安装和使用

## install

```sh
# scoop
scoop install alacritty
```

## Usage

手动建立配置文件

- macOS & Linux: ~/.config/alacritty/alacritty.yml
- Windows: %APPDATA%\alacritty\alacritty.yml

### 常用快捷键

| 常用快捷键                 | 动作           |
| -------------------------- | -------------- |
| `alt` + `enter`            | 最大化         |
| `ctrl` + `shift` + `space` | toggle vi mode |
| `ctrl` + `shift` + `space` | 最大化         |
