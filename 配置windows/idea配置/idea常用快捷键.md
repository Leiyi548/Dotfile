# idea 常用快捷键

|         键位         |      动作      |
| :------------------: | :------------: |
| `ctrl` +`shift` +`f` | 全局查找字符串 |
|     `ctrl` +`n`      |     查找类     |
| `ctrl` +`shift` +`n` |    查找文件    |
|     `ctrl` +`r`      | 全局替换字符串 |
