# lazygit安装和使用

## Install

```powershell
# Add the extras bucket
scoop bucket add extras

# Install lazygit
scoop install lazygit 
```

## Usage

### 配置文件存放路径

windwos默认路径:
%APPDATA%\lazygit\config.yml
> 其中%APPDATA% 这个代表的路径 C:\Users\username\AppData\Roaming
> 我的用户名是9
> 完整路径 C:\Users\9\AppData\Roaming\lazygit\config.yml
