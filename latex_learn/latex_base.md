# latex 在 markdown 中的使用

## 加减乘除

- 加:$+$
- 减:$-$
- 乘:$\times$
- 除:$\div$
- 正负号:$\pm$

## 上下标

- 上标:$x^{n}$
- 下标:$x_{n}$

## 箭头符号

- 上箭头:$\uparrow$
- 下箭头:$\downarrow$
- 双线上箭头:$\Uparrow$
- 双线下箭头:$\Downarrow$
- 右箭头:$\rightarrow$
- 左箭头:$\leftarrow$
- 双线右箭头:$\Rightarrow$
- 双线左箭头:$\Leftarrow$
- 双线右长箭头:$\Longrightarrow$
- 双线左长箭头:$\Longleftarrow$
